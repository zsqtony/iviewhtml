# 87ui
#### 87UI是为了方便前端开发者，在开发后台管理系统时，能简单、快速的搭建好一个ui界面简洁美观的一个例子。
#### 开发者只需"clone"拉取下来安装依赖，即可往里面装填页面组件开发。
